## 🚀 项目简介

**wind** 是一款基于 Go 语言开发的现代化加密工具库，专注于数据安全，提供简单易用的接口和高性能加密能力，适合各种安全需求场景。

### 🌟 **核心特性**

✅ 支持 RSA 和 AES 多种加密算法。  
✅ 提供签名与验证，确保数据完整性与合法性。  
✅ 支持证书生成与管理，方便密钥管理。  
✅ 高度可配置，支持自定义填充和编码格式。  
✅ 兼容多平台，满足跨平台开发需求。
